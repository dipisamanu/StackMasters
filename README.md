## Requisiti di Sistema
Per lavorare a questo progetto, assicurarsi di avere:
1. **XAMPP** installato
2. **Database:** MySQL
3. **Versione PHP:** Minimo 8.1 o 8.2 (da PhPStorm)
4. **Composer:** Installato nella cartella del progetto (/composer)
